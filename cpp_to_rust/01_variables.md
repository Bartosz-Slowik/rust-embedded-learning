# Constants
## C++
```c++
const int x = 0;
```

## Rust
```rust
// `let` is a keyword used for declaring variables.
let x: i32 = 0;
```

# Variables
## C++
```c++
int x = 0;
```

## Rust
```rust
// `mut` means mutable - liable to change
let mut x: i32 = 0;
```

# Type inference
## C++
```c++
auto x = 0;
```

## Rust
Rust infers types by default.
```rust
let x = 0;
```
